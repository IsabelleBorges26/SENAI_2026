Anotações aula Back-End



1. npm init -y: comando para inicializar projeto e dar todas as permições
2. após isso, muda para server,js no package.json em main
3. instalar as dependências: express, cors, mysql2, dotenv - "npm i express cors mysql2 dotenv"
4. criar arquivo server.js
5. instalar "npm i nodemon" e rodar "npx nodemon"
